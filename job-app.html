项目介绍

零工有靠是一款面向灵活用工群体的求职与兼职服务平台，以「米米」为品牌 IP 形象，为用户提供岗位发现、地图找工、在线报名、签到奖励等一站式服务。

视觉体系采用清新明亮的嫩绿色作为主色调，搭配圆润可爱的 3D IP 图标与角色，传递轻松、可信赖、充满活力的品牌调性。设计贯穿从首页信息架构到核心业务链路的全流程，兼顾用户转化效率与情感化体验。

首页设计

首页采用P1 兼职指南 → P2 金刚区/磁片区 → P3 双列 Feed 流推荐的三层信息架构。头部 P1 区域采用明亮清新的主题绿色，视觉设计简洁明了，营造轻松活泼的兼职氛围。「多云转晴」的天气隐喻暗示找到好工作的美好预期。

针对有明确目标的用户，P2 设置常用功能入口对首屏进行分流。视觉层采用与业务相关的萌系能量 3D 质感图标，结合前瞻定义的视觉风格加强视觉冲击力。P3 双列 Feed 流根据用户偏好进行个性化岗位推荐。

岗位报名

基于LBS 地理位置的岗位发现流程，用户可通过地图视图直观浏览周边工作机会。三屏交互流程覆盖：发现岗位详情 → 查看岗位信息流 → 填写个人意愿报名，每一步都经过体验优化，降低用户操作门槛。

签到活动

签到页是产品中运营属性偏重的一页。元素层作为视觉核心内容的主要载体，直接承接粗突击主题、吸引用户目光的重任，风格上与整体页面图标的萌系风格保持一致。

P1 积分进度条：阶梯现金奖励拉动连续签到，分层展示每日收益。递进式红包进度条一目了然，红绿色块反差鲜明，标注分值每日奖励，「最高」标签放大收益吸引力。

推荐活动 & 奖励兑换：针对不同用户群体（日结/长期/高薪）做差异化内容推荐，结合签到、分享裂变等运营玩法。「推荐活动」弹窗 + 「奖励兑换」抽奖转盘形成完整的用户增长闭环。

IP 角色延展

围绕品牌 IP 打造完整角色矩阵，每个角色对应一个使用场景或用户状态：

兼职招募、咖啡店员、外卖配送、超市收银（第一排）对应核心求职场景；家教辅导、活动协助、宠物照料、仓库打包（第二排）覆盖生活化灵活用工场景。统一的 Q 版嫩绿星形头部形象配合差异化动作表情与道具，强化品牌辨识度与情感连接，让用户产生亲切感与代入感。
